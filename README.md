# 1HappyEnd Demo

Bu klasor, 1HappyEnd icin hazirlanmis calisan ilk demo prototipidir.

## Calistirma

Yerel sunucu su adreste calisiyor:

```text
http://127.0.0.1:4173/
```

Giris sayfasi:

```text
http://127.0.0.1:4173/account/login/
```

Demo admin bilgileri:

```text
Kullanici adi: admin
Sifre: 123456
```

Demo ziyaretci bilgileri:

```text
Kullanici adi: visitor
Sifre: 123456
```

Demo reklam veren bilgileri:

```text
Kullanici adi: advertiser
Sifre: 123456
```

## Hazir Akislar

- 18+ yas dogrulama ekrani
- Ilan arama, hizmet, konum ve kategori filtreleri
- Musait ilan filtresi
- Ilan detay penceresi
- Favori ve destek talebi demo bildirimleri
- Admin girisi
- Ziyaretci girisi ve kaydi
- Reklam veren girisi ve kaydi
- Rol bazli hesap ayrimi
- Cuzdan, HE Coin, EUR/USD odeme hazirligi
- Paket secimi ve mock checkout akisi
- Coklu dil secimi: TR, EN, NL, DE, FR, ES, IT, PT, RU, AR
- Admin dashboard istatistikleri
- Ilan ekleme, duzenleme ve silme
- One cikan ilan/vitrin kontrolu
- Telefon ve tablet icin responsive arayuz
- Markali premium tasarim: gercek logo, kirmizi hero arka plani, guven kartlari ve directory tipi filtre paneli
- Modern Tailwind/Next esintili marketplace UI: metrik kartlari, temiz SaaS panelleri, mavi aksanlar ve dashboard hissi

Veriler tarayicinin localStorage alaninda tutulur; demo sirasinda sayfa yenilense bile eklenen ilanlar korunur.
